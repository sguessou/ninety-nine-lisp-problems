### Working with lists
#### Problem #1:
*Find the last box of a list.*
```
CL-USER> (my-last '(a b c d))
CL-USER> (D)
``` 
#### Problem #2:
*Find the last but one box of a list.*
```
CL-USER> (my-but-last '(a b c d))
CL-USER> (C D)
```
